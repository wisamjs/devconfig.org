Yeoman Open source Tooling Hackathon

What's Yeoman?

Yeoman helps you kickstart new projects, prescribing best practices and tools to help you stay productive.

The Yeoman workflow is comprised of three types of tools for improving your productivity and satisfaction when building a web app: the scaffolding tool (yo), the build tool (Grunt, Gulp, etc) and the package manager (like Bower and npm.

Why an Open source Hackathon?

Whatever your tool of choice is, and regardless of what Open source libraries you build your projects with, We all love Open source Software!

These Open source projects are changing the software world into better place for everybody to create, share and build amazing products. You can help too!

Hackathons are great at getting the community together, share learnings and build meaningful products and services. We want to bring the spirit of product building in Hackathons to the Open Source world so that we can all benefit from improving the tools and libraries we use everyday.

What to expect:

We'll be focusing on yo, Grunt, and Bower as the three major tools, and with the help of the maintainers we'll tackle some open issues, new features and expand documentation and tutorials.

Build new features & plugins

  • contribute to planned milestones
  • suggest new features
  • build new plugins

Tackling bugs & issues

  • respond/address github issues
  • write unit tests
  • reach 100% code coverage!

Documentations & tutorials

  • tranaslate documentation into different languages
  • write step-by-step tutorials

Requirements:

  • JavaScript & NodeJS
    Intermediate to Expert level in JavaScript is required, If you've never worked with any of the tools, but have a good understand of NodeJS, this would be a good opportunity to learn.

  • BYOL
    Bring your own laptop, WiFi is available.

  • RTFM
    start by reading each of the tools README.md & Contributing.md: (yo, Grunt, Bower)

  • RTFC
    get friendly and familiar with the source code ahead of time: (yo, Grunt, Bower)
